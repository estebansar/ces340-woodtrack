# ces340-woodtrack

This is my final project for the CSE 340 Web Backend Development class.

The idea of this project is to create a simple website where users can share their woodworking projects and keep track of custom build requests. I want to use this project to practice what I learned in this course, like working with Express, connecting to a PostgreSQL database, using EJS, and building login and role based features.

Users will be able to create an account, add their own projects, comment on projects, and submit requests. There will also be an admin area to manage content and users.

Right now the project is still in the early stages and more features will be added as I continue working on it.

Technologies planned:
- Node.js  
- Express  
- EJS  
- PostgreSQL  
- Render
